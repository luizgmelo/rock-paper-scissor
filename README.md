# Rock Paper Scissors

## What is this project:
- Game Rock Paper Scissor versus computer.

## What learn here:
- Python
  - Loop (While), Conditions
  - Break and Continue
  
## How use it:
- Clone this repository to your machine.
- Open folder on terminal.
- Execute: python main.py
